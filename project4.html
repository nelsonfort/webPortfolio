Sistema de Riego Inteligente para Granjas

Sobre Este Proyecto

This project delivers a comprehensive IoT-based solution for automating farm irrigation, designed to optimize water usage and crop health. The system comprises a family of distributed nodes equipped with sensors to monitor soil moisture and other environmental parameters. These nodes are built around the ESP32 microcontroller, utilizing FreeRTOS for efficient task management.

Connectivity is flexible, supporting both long-range Lorawan communication (up to SF12 RX2) for wide area coverage and Cat-M1 cellular for reliable internet access. Devices communicate using MQTT protocol to a backend hosted on AWS, which includes a Linux virtual machine running ChirpStack (for Lorawan), Node-RED (for data processing and logic) and a custom WebApp for user interaction and control.

The system controls various types of motorized gates to direct water flow to different sections of the farm based on sensor data and user-defined schedules. Control is possible remotely via the web application, or locally through an integrated user interface on the devices, providing robustness and ease of use.

Componentes del Sistema:

  • Nodos sensor/actuador basados en ESP32.
  • Comunicación LoRaWAN y Cat-M1.
  • Protocolo MQTT para transferencia de datos.
  • Backend en AWS con ChirpStack, Node-RED, WebApp.
  • Control de compuertas motorizadas.
  • Interfaces de usuario web remota y local.
